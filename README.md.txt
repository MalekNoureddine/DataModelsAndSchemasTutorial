# Data Models and Schemas Tutorial

This repository contains solutions to the "Review of Data Models and Schemas" tutorial.

## Exercise 1: Car Sales Company
- Includes an ER Diagram, database schema, and SQL queries for various tasks.
- Files:
  - `ERDiagram.png`: ER Diagram.
  - `DatabaseSchema.sql`: SQL script to create the database schema.
  - `Queries.sql`: SQL queries for the exercise.

## Exercise 2: CINEMA Database
- Includes SQL queries to answer questions related to the CINEMA database schema.
- Files:
  - `Queries.sql`: SQL queries for the exercise.

## How to Use
- Clone this repository: `git clone <repo-url>`
- Open the `.sql` files in your preferred SQL editor to execute the queries.
